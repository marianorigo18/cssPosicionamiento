La propiedad clear se usa para controlar el comportamiento de los elementos que siguen a los elemetos flotados, es decir no dejandolos flotar hacia donde nosotros hagamos el float
¿Cuando es compatible clear?
solamente cuando le declaramos un valor que es igual a la orientacion del float, osea si float es right entonces clear tenemos que asignarle right si no no funcionara.
ejemplo, la imagen de arriba tiene un float left que se hace ver como el cuadro rojo, pero cuando le damos al parrafo un clear left queda como por defecto, en este caso parece que no hay cambios pero cuando haya muchos elementos esta propiedad sera muy util
